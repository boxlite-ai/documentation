BoxRun wraps the BoxLite micro-VM runtime into a batteries-included platform: a REST API server, CLI, web dashboard, and SDKs for Python and Rust — all in a single binary.

<Tip>
  Want to embed sandboxing as a library instead? See the [BoxLite SDK quickstarts](/getting-started/quickstart-python).
</Tip>

## Install

<Note>
  **Requirements:** macOS Apple Silicon (M1+). Linux support coming soon.
</Note>

```bash
curl -fsSL https://boxlite.ai/boxrun/install | sh
```

## Quick start

```bash
# One command: create a VM and drop into a shell
boxrun shell ubuntu

# Or step by step
boxrun create ubuntu:24.04 --name dev
boxrun attach dev

# Run commands non-interactively
boxrun exec dev -- uname -a

# Mount a host directory into the VM
boxrun create ubuntu:24.04 --name work -v ~/project:/root/project

# Copy files in and out
boxrun cp ./data.csv dev:/root/data.csv
boxrun cp dev:/root/results.csv ./results.csv

# Lifecycle
boxrun stop dev     # preserves disk
boxrun start dev    # restart
boxrun rm dev       # destroy
```

<Tip>
  The server starts automatically on your first command — no need to run `boxrun serve` manually.
</Tip>

## Python SDK

```bash
pip install boxrun
```

```python
import asyncio
from boxrun import BoxRunClient

async def main():
    async with BoxRunClient() as client:
        # Create a sandbox
        box = await client.create("ubuntu:24.04", name="dev")

        # Run a command
        result = await box.exec(["echo", "hello from BoxRun"])
        print(f"Exit code: {result.exit_code}")

        # Stream output in real time
        async for event in box.exec_stream(["apt-get", "update"]):
            if event.type == "log":
                print(event.data, end="")

        # Upload and download files
        await box.upload("./script.py", "/root/script.py")
        await box.download("/root/output.json", "./output.json")

        await box.remove()

asyncio.run(main())
```

For one-shot tasks, use `client.run()` — it creates a box, runs the command, returns output, and destroys the box:

```python
result = await client.run("ubuntu:24.04", ["python3", "-c", "print(42)"])
print(result.stdout)  # "42\n"
```

## Web dashboard

A built-in web UI is available at `http://localhost:9090/ui` when the server is running. You can view all boxes, start/stop/delete them, and see execution history.
