BoxLite ships with nine Python examples in the repository. Each is a standalone script you can run after building the Python SDK.

<Tip>
  New to BoxLite? Start with the [Tutorials](/tutorials/index) for guided walkthroughs. This page is a quick reference for the runnable examples in the repo.
</Tip>

## Prerequisites

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/boxlite-ai/boxlite.git
    cd boxlite
    ```
  </Step>
  <Step title="Build the Python SDK">
    ```bash
    make dev:python
    ```
  </Step>
</Steps>

## Examples

| # | File | Description | Run |
|---|------|-------------|-----|
| 1 | `simplebox_example.py` | Command execution, stdout/stderr, env vars, working dirs | `python examples/python/simplebox_example.py` |
| 2 | `codebox_example.py` | AI code execution with auto package install | `python examples/python/codebox_example.py` |
| 3 | `browserbox_example.py` | Browser automation via CDP | `python examples/python/browserbox_example.py` |
| 4 | `computerbox_example.py` | Desktop automation (mouse, keyboard, screenshots) | `python examples/python/computerbox_example.py` |
| 5 | `lifecycle_example.py` | Box start, stop, restart, state inspection | `python examples/python/lifecycle_example.py` |
| 6 | `list_boxes_example.py` | Enumerate and inspect active boxes | `python examples/python/list_boxes_example.py` |
| 7 | `cross_process_example.py` | Share boxes across processes | `python examples/python/cross_process_example.py` |
| 8 | `interactivebox_example.py` | Interactive TTY shell sessions | `python examples/python/interactivebox_example.py` |
| 9 | `native_example.py` | Low-level Rust API from Python | `python examples/python/native_example.py` |
