Lightweight micro-VM sandboxes for AI agents and secure code execution. Hardware isolation, sub-second boot, zero daemon — runs on macOS (Apple Silicon), Linux (KVM), and Windows (WSL2).

## Quick start

<Tabs>
  <Tab title="BoxRun (CLI)">
    ```bash
    # Install
    curl -fsSL https://boxlite.ai/boxrun/install | sh

    # One command — create a VM and drop into a shell
    boxrun shell ubuntu
    ```
  </Tab>
  <Tab title="BoxLite (Python)">
    ```bash
    pip install boxlite
    ```

    ```python
    import asyncio
    import boxlite

    async def main():
        async with boxlite.SimpleBox() as box:
            result = await box.exec("echo", "Hello from BoxLite!")
            print(result.stdout)

    asyncio.run(main())
    ```
  </Tab>
  <Tab title="BoxLite (Node.js)">
    ```bash
    npm install boxlite
    ```

    ```typescript
    import { SimpleBox } from 'boxlite';

    const box = new SimpleBox({ image: 'alpine:latest' });
    try {
      const result = await box.exec('echo', 'Hello from BoxLite!');
      console.log(result.stdout);
    } finally {
      await box.stop();
    }
    ```
  </Tab>
</Tabs>
