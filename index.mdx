BoxLite lets you spin up lightweight Linux VMs and run OCI containers inside them — in milliseconds, from your own code, with no daemon and no root. Every sandbox is a real microVM with its own kernel, providing hardware-level isolation that containers can't match.

Runs on macOS (Apple Silicon), Linux (KVM), and Windows (WSL2).

## What is BoxLite?

<CardGroup cols={3}>
  <Card title="Embedded" icon="cube">
    A library, not a service. Import it in Python, Node.js, Rust, or C — no daemon, no Docker, no external dependencies.
  </Card>
  <Card title="Hardware isolation" icon="shield-halved">
    Each sandbox is a real microVM with its own Linux kernel. Full hardware isolation via KVM or Apple Hypervisor.framework.
  </Card>
  <Card title="Lightweight" icon="feather">
    VMs boot in ~150ms with minimal overhead. Run hundreds of sandboxes on a single host.
  </Card>
</CardGroup>

## Quick start

<Tabs>
  <Tab title="Python">
    ```python
    # pip install boxlite
    import asyncio
    import boxlite

    async def main():
        async with boxlite.SimpleBox() as box:
            result = await box.exec("echo", "Hello from BoxLite!")
            print(result.stdout)

    asyncio.run(main())
    ```
  </Tab>
  <Tab title="Node.js">
    ```typescript
    // npm install @boxlite-ai/boxlite
    import { SimpleBox } from '@boxlite-ai/boxlite';

    const box = new SimpleBox({ image: 'alpine:latest' });
    try {
      const result = await box.exec('echo', 'Hello from BoxLite!');
      console.log(result.stdout);
    } finally {
      await box.stop();
    }
    ```
  </Tab>
</Tabs>

## BoxRun — the platform layer

**BoxRun** wraps BoxLite into a ready-to-use platform: CLI, REST API, web dashboard, and Python SDK — all in a single binary. No code required to get started.

```bash
curl -fsSL https://boxlite.ai/boxrun/install | sh
boxrun shell ubuntu
```

BoxRun is built on BoxLite. Choose BoxRun when you want a server you can talk to over HTTP or CLI. Choose BoxLite when you want to embed sandboxing directly in your application code.

## SDKs and platforms

| SDK | Install | Status |
|-----|---------|--------|

| macOS | Apple Silicon | macOS 12+ |
| Linux | x86_64 / ARM64 | KVM enabled |
| Windows | WSL2 | KVM support |
